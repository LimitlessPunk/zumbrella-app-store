# Zumbrella App Store

Community app store for [Umbrel](https://umbrel.com) providing Zcash infrastructure apps.

## Apps

### Zcash Node

Run a Zcash full node powered by Zebra (zebrad) with a light wallet server (lightwalletd) and web dashboard.

- Full blockchain validation and sync
- Light wallet server for Zashi, YWallet, ZingoWallet
- Web dashboard with sync progress, peer info, and network stats
- Tor hidden services for private access

## Installation

1. Open your Umbrel dashboard
2. Go to **App Store** > **...** (three dots) > **Community App Stores**
3. Add URL: `https://github.com/LimitlessPunk/zumbrella-app-store`
4. Find "Zcash Node" and click **Install**
